# Amazon Clone Website

This is a simple **Amazon Clone website** built using **HTML, CSS, and JavaScript**.
The project replicates the basic layout and design of the Amazon homepage.

## 🚀 Live Demo

View the website here:
https://amazon-clone80.netlify.app/

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📌 Features

* Responsive homepage layout
* Product sections similar to Amazon
* Navigation bar and search bar
* Clean and modern UI design

## 📂 Project Structure

* index.html
* style.css
* script.js
* images/

## 📖 Purpose of the Project

This project was created for **learning front-end web development** and improving skills in HTML, CSS, and JavaScript.
